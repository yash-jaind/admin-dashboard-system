# Admin Dashboard System

A role-aware admin dashboard built with React, Node.js, Express, and MongoDB.

## Features

- Role-Based Access Control (RBAC)
  - Admin
  - Manager
  - Customer

- Authentication & Authorization
  - JWT-based login
  - Protected routes
  - Permission-based UI rendering

- Dashboard Analytics
  - Total Users
  - Total Products
  - Revenue Statistics
  - Order Tracking
  - Growth Charts

- User Management
  - Add Users
  - Edit Users
  - Delete Users
  - Search & Pagination

- Product Management
  - Create Products
  - Update Products
  - Delete Products
  - Inventory Tracking

- Notification System
  - Read/Unread Notifications
  - Notification Bell
  - Activity Updates

- Responsive Admin Interface
  - Sidebar Navigation
  - Modern Dashboard Layout
  - Mobile-Friendly Design
